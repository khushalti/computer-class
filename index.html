Class 1: Introduction To Python

Python

Install and run "python" to start the interpreter. Try basic math statements.

>>> 2+2
4
>>> 5 * 20
100
>>> 10 / 3
3

Variables

Values can be given names (Variables) and can be used instead of numbers.

>>> my_marks / total_marks * 100

Lists

Values can be maintained in a list and accessed by its "index".

>>> my_list = [23, 4, 6, 'harry']
>>> my_list[0]
23

Functions

A function is defined to run a group of statements and return a value.

>>> def what_is_my_percent(my_marks, total_marks):
      return my_marks / total_marks * 100
>>> what_is_my_percent(273.0, 300)
91.0

Class 2: Loops and Modules

Loops

A loop can run a group of statements many times.

>>> def add_marks(marks):
      total_marks = 0
      for mark in marks:
        total_marks = total_marks + mark
      return total_marks
>>> add_marks([20, 30, 40, 10])
100

String Concatenation

Letters (strings) can be joined to make sentences.

>>> def say_hello(students):
      student_names = ''
      for student in students:
        student_names = student_names + ' ' + student + ','
      return 'Hello' + student_names[:-1] + '. How are you?'
>>> say_hello(['Harry', 'Hermione', 'Ron'])
Hello Harry, Hermione, Ron. How are you?

Modules

Functions and variables can be put in files called "modules". The can be imported using the import statement.

>>> import wizard
>>> wizard.add_marks([20, 30, 40])

Class 3: Classes and Conditions

Classes

Classes can be used to define concepts with many properties. Concepts can then be made into objects.

class House:
    ability = None

class Student:
    ability = None

gryffindor = House()
gryffindor.ability = 'bravery'

harry = Student()
harry.ability = 'bravery'

Inspection

Modules, objects, classes can be inspected to discover its inner properties.

>>> import sortinghat
>>> dir(sortinghat)
['House', 'Student', '__builtins__', '__doc__', '__file__', '__name__', '__package__', 'gryffindor', 'harry']

Conditions

if statement can run a group of statements based on a condition.

def sort(student):
    for house in all_houses:
        if house.ability == student.ability:
            if house.students is None:
                house.students = []

            house.students.append(student)
            print student.name + ' goes to ' + house.name + '!'

Class 5: HTML

index.html

HTML is a file with text that is decorated with tags like <h1> etc.

<html>
<h1>Shishuvan Computer Club</h1>
<p>Welcome to the Shishuvan Computer Club</p>
</html>

Style

Tags can have style attribute that sets its spacing, font, color etc.

<html class='font-family: Joker; background-color: yellow; padding: 30px;'>
<h1 style='color: red'>Shishuvan Computer Club</h1>
<p>Welcome to the Shishuvan Computer Club</p>
</html>

<div> Block

Rectangular blocks can be defined using the <div> tag.

<html class='font-family: Joker; background-color: yellow'>
<h1 style='color: red'>Shishuvan Computer Club</h1>
<p>Welcome to the Shishuvan Computer Club</p>
<div style='height: 10px; color: blue;'></div>
</html>
